# Landing Page - Glory Dashboard

## Público Objetivo
Desarrolladores y usuarios técnicos que buscan herramientas de productividad avanzadas.

---

## Estructura

### 1. Hero
- **Headline:** Frase corta y directa. Sin adornos.
- **Subheadline:** Una línea describiendo qué hace.
- **CTA:** Botón "Probar gratis" o "Ver demo"
- **Visual:** Screenshot limpio del dashboard

### 2. Features (3-4 máximo)
| Feature      | Descripción corta                      |
| ------------ | -------------------------------------- |
| Hábitos      | Seguimiento con mapa de calor y rachas |
| Tareas       | Drag & drop, subtareas, adjuntos       |
| Cifrado E2E  | AES-256-GCM, tus datos son tuyos       |
| Open Source* | Código auditable                       |

*Si aplica*

### 3. Cómo Funciona
3 pasos máximo:
1. Registra tu cuenta
2. Configura tus hábitos/tareas
3. Visualiza tu progreso

### 4. Stack/Tech (opcional)
Lista simple para credibilidad técnica:
- React + TypeScript
- WordPress REST API
- Zustand
- Vite

### 5. CTA Final
Repetir el botón principal.

---

## Notas de Diseño
- **Sin animaciones excesivas.** Transiciones sutiles.
- **Fondo oscuro** preferido (terminal style).
- **Tipografía monospace** para headers (opcional).
- **Sin testimonios falsos.** Solo mostrar si hay reales.
- **Sin pricing complejo.** Free/Pro y listo.

- Contadores animados ("10,000+ usuarios")
- Pop-ups de newsletter
- Textos largos explicativos
- Imágenes de stock
